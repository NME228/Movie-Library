Автор: Нужин Матвей
Описание: GUI-приложение «Movie Library» - приложение, которое помогает добавлять фильм в личную кинотеку, указывая год выпуска, жанр, рейтинг(1-10), а также искать добавленные в свою кинотеку фильмы с помощью фильтрации по жанру и году выпуска
Примеры использования: Можно добавлять любимые фильмы в свою кинотеку, чтобы их было легко найти, указав жанр и год выпуска фильма
